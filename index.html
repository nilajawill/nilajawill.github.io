The Music Bin

The Music Bin Webapp is a creative solution to an interesting compliant from a friend. For anyone who has saved Youtube videos in playlist, you may have come across certain videos disappearing due to copyright issues or country regulations. These, as my friend stated, creates "holes" in your playlist.

Therefore, to ensure that songs can always be found, I created a Rails App that stores basic information about songs - the title and artist of the song, it's Youtube link (if applicable), and an option to add notes. With The Music Bin Webapp, you will never lose a song again!

Hibernate Overview
The Hibernate is an ORM framework simplifies DB access by representing data as simple Java objects (POJO). POJO are accessed by the session manager. Hibernate can be used in any Java application (Spring, Serlvet, etc).
It takes care of our database access through mapping and configuration, hidding the database schema, adding more focus on business logic. Query results are objects instead of SQL result sets. Persistence is achieved by mapping objects to tables in the database.
